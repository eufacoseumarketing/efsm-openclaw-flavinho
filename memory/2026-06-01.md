## Problemas Encontrados e Resolvidos

### 1. Boot Linux quebrado
- **Sintoma**: `READ FPDMA QUEUED` + `ATA bus error` + `COMRESET failed`
- **Inicial**: Parecia HD morto — kernel não conseguia ler disco
- **Depois**: Chegou no initramfs BusyBox — `/dev/sda7: UNEXPECTED INCONSISTENCY`
- **Causa real**: Partição Linux corrompida (provavelmente desligamento forçado)
- **Windows**: Boot normal, dual boot OK
- **Disco**: SSHD OK (wmic Status=OK)

### 2. WiFi offline (Intel Wireless-N 7260)
- **Sintoma**: WiFi não aparece, Modo Avião cinza
- **Diagnóstico**: Dispositivo oculto com Código 45 ("não conectado ao computador")
- **Tentativas**: Fn+F7 (não funcionou — driver hotkey Lenovo ausente), reinstalar driver, desinstalar
- **Pós-desinstalar**: Placa sumiu completamente — nem como oculto
- **BIOS**: Sem toggle WLAN (Configuration tab não tem opção)
- **Conclusão**: Placa fisicamente solta ou morta — precisa abrir e reseatar/trocar (~R$50)
- **Workaround**: Ethernet cabo funcionando

### Outras lições
- 📡 WiFi Código 45 em dual boot = provável toggle físico desligado pelo Linux (Fn+Fx)
- 🔧 Lenovo Flex 2: Fn+F7 é WiFi, mas precisa do driver Lenovo Energy Management
- 💻 BITS (`bitsadmin`) é ideal pra downloads grandes — não trava API, retoma sozinho
- ⚠️ `run` API = SYSTEM = sem GUI. Pra apps visuais: `schtasks` ou orientar usuário
- 🖥️ Foco roubado por popup (HP Deskjet) = Enter vai pro lugar errado
- 🐧 Linux quebado em dual boot: `fsck -y /dev/sda7` no initramfs ou reinstalar limpo
- 🔑 `key: "win"` não funciona na API v1.5.1 — usar outras teclas
