## Lições Aprendidas (🔥 REGRAS DEFINITIVAS)

### 1. Atualização de drivers Dell
- **SEMPRE usar ferramentas automáticas primeiro** (SupportAssist, Intel DSA)
- **NUNCA caçar driver por driver no site da Dell** — site bloqueia VPS (403), direto dá 404
- Ordem: SupportAssist → Intel DSA → Windows Update → site do fabricante (último recurso)
- SupportAssist download: `https://downloads.dell.com/serviceability/catalog/SupportAssistInstaller.exe`
- Instalação silenciosa: `/VERYSILENT`

### 2. BIOS = ALTO RISCO
- ⚡ SEMPRE avisar antes e pedir confirmação explícita
- Backup completo ANTES de flashar BIOS
- Não desligar/interromper durante o flash
- O reboot pode levar 3-5 minutos (tela preta)

### 3. Backup
- ✅ Copiar → Compactar → INSTRUIR cliente a salvar FORA da máquina
- 🚫 NUNCA salvar backup na VPS/ambiente PC Resolve
- ⚠️ Exceção só com autorização explícita do Zanatto (e deletar depois)
- Backup no mesmo disco NÃO é backup

### 4. Navegação e UI
- 📜 Sempre maximizar + scrollar antes de agir (ver TODOS os itens)
- 🖥️ `run` API = SYSTEM, não lança GUI no usuário
- ✅ Abrir apps no contexto do usuário: Start Menu + `type` nome + Enter
- 🎯 Click: coordenadas precisas, confirmar com screenshot pós-click

### 5. API PC Resolve (validado hoje)
- `key`: Enter, pgdn, pgup, esc, tab, down, up — teclas reais
- `type`: Ctrl+A + texto (substitui, não anexa)
- `open-url`: prefere Chrome
- `click`: dois chamados = double-click
- Screenshot: ~2.7s, ~67KB (full VirtualScreen)
